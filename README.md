# fgacic (Vite + React)

Centered single-screen UI. Commands: `yarn`, `yarn dev`, `yarn build`.

Uses Vite 5 for compatibility with Node 20.x and 21.x (Vite 8 expects Node 20.19+ or 22.12+).

## Front page showcase

The headline **Bula Gej** uses a ~15s staged intro: layered glow and conic wash ramping up, sparks, an elliptical ring burst near the peak, staggered letter pops with tracking opening, a one-pass scan shimmer, brief chromatic fringe on the line, then softer ambient motion (gradient drift, float, ring/breathing accents). With `prefers-reduced-motion`, long motion is skipped for a static hero look.
